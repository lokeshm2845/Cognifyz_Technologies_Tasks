## 📌 Overview

This repository contains the solutions for the **Software Development Internship** program at **Cognifyz Technologies**. 

As per the internship guidelines, I have completed **4 out of 6** assigned tasks. These projects focus on core programming concepts, including conditional logic, loop structures, CRUD operations, file handling, and web scraping.

- **Language**: Python 3.8+

## 🚀 Getting Started (Setup & Installation)

Follow these steps to run the projects locally:

1. **Clone the repository** (or download the ZIP):
   ```bash
   git clone https://github.com/your-username/cognifyz-internship-tasks.git
   cd cognifyz-internship-tasks
   ```

2. **Install the required dependencies** (only needed for Task 6):
   ```bash
   pip install requests beautifulsoup4
   ```

3. **Run any task**:
   Open your terminal and execute the specific Python file.
   ```bash
   python task1_game.py
   python task2_pattern.py
   python task3_crud.py
   python task4_temp.py
   python task5_crud_file.py
   python task6_scraper.py
   ```
